Passo 0

Introducao: Java e Spring Boot

Antes de entrar no Docker e no CRUD, alinhamos a base tecnologica do projeto: o que e Java, quando surgiu, o que e Spring Boot e por que estamos usando essa stack.

O que e Java

  • Linguagem orientada a objetos e multiplataforma (JVM).
  • Criada na Sun Microsystems, com lancamento inicial em 1995.
  • Hoje mantida pela Oracle e amplamente usada em back-end corporativo.

O que e Spring Boot

  • Framework do ecossistema Spring para acelerar APIs Java.
  • Projeto da Pivotal, primeira versao em 2014.
  • Usamos para reduzir configuracao manual e focar em regra de negocio.
Neste treinamento, vamos aplicar Java + Spring Boot em uma API real com Docker, H2 e testes sequenciais. Objetivo pratico: sair com o CRUD inicial de produtos funcionando e documentado.

Fixação rápida: por que usar seeds?

  • Garante estado inicial previsível para cada execução.
  • Reduz flakiness em testes de listagem e integração.
  • Facilita reproduzir cenários no time inteiro.
